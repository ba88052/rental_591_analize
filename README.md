# 591租屋網分析與爬蟲API 

此專案使用爬蟲爬取591租屋網的租屋資料，並將Web API架設於GCP上。


## Get Start
### Installation
此專案下載至桌面後，使用以下指令安裝必要套件。

    pip3 install -r requirements.txt
    #or
    pip install -r requirements.txt

### Running the Project
套件安裝成功後，即可開始執行本專案。
    
    python main.py

This app will be running on localhost:5000.
